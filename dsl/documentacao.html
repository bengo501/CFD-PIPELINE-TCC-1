introdução

o que é este projeto?

pipeline automatizado para simulação cfd de leitos empacotados usando uma dsl personalizada, blender para geometria 3d e openfoam para simulação.

dsl customizada

linguagem .bed para descrever parâmetros do leito de forma intuitiva

geometria 3d

geração automática de modelos 3d realistas no blender com física

simulação cfd

integração com openfoam para análise de escoamento

containerizado

docker compose para reprodutibilidade total

arquitetura do sistema

usuário
define parâmetros via wizard ou arquivo .bed
↓
compilador antlr
valida sintaxe e gera arquivo.bed.json
↓
blender (headless)
cria geometria 3d com física de partículas
↓
openfoam
simulação cfd do escoamento
↓
visualização
dashboard web com resultados

instalação

pré-requisitos

  • python 3.8+ - linguagem principal
  • java 11+ - para executar antlr
  • blender 3.0+ - geração de geometria
  • git - controle de versão

instalação rápida

1. clonar repositório

git clone https://github.com/bengo501/CFD-PIPELINE-TCC-1.git
cd CFD-PIPELINE-TCC-1

2. instalar dependências python

pip install antlr4-python3-runtime

3. gerar parser antlr

cd dsl
java -jar antlr-4.13.1-complete.jar -Dlanguage=Python3 -o generated grammar/Bed.g4

pronto! agora você pode usar o wizard:

python bed_wizard.py

uso rápido

modo 1: wizard interativo

python bed_wizard.py
# escolher opção 3 (modo blender)
# preencher parâmetros
# aguardar geração

modo 2: editar template

python bed_wizard.py
# escolher opção 2
# editar arquivo .bed
# compilar e executar

modo 3: criar arquivo manualmente

// arquivo: meu_leito.bed
bed {
    diameter = 0.05 m;
    height = 0.1 m;
    wall_thickness = 0.002 m;
}

particles {
    kind = "sphere";
    count = 100;
    diameter = 0.005 m;
    density = 2500.0 kg/m3;
}

packing {
    method = "rigid_body";
    gravity = -9.81 m/s2;
}

depois compilar:

python compiler/bed_compiler_antlr_standalone.py meu_leito.bed

parâmetros principais

seção bed (geometria)

parâmetro range descrição
diameter 0.01 - 2.0 m diâmetro interno do leito
height 0.01 - 5.0 m altura do leito
wall_thickness 0.0001 - 0.1 m espessura da parede

seção particles

parâmetro range descrição
kind sphere, cube, cylinder formato das partículas
count 1 - 10000 quantidade de partículas
diameter 0.0001 - 0.5 m diâmetro das partículas
density 100 - 20000 kg/m³ densidade do material

seção packing

parâmetro range descrição
method rigid_body método de empacotamento
gravity -50 a 50 m/s² aceleração gravitacional
substeps 1 - 100 precisão da simulação

dica: use o sistema de ajuda!

no wizard, digite ? em qualquer campo para ver informações detalhadas, ranges permitidos e exemplos práticos.

workflow típico

1. definir parâmetros

  • abrir wizard: python bed_wizard.py
  • escolher modo blender (opção 3)
  • preencher parâmetros com ajuda contextual

2. gerar geometria

  • wizard compila automaticamente o .bed
  • executa blender em modo headless
  • salva modelo em generated/3d/output/

3. visualizar resultado

  • abrir arquivo .blend no blender
  • verificar geometria e empacotamento
  • exportar para outros formatos se necessário
